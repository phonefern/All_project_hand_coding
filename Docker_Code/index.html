Designing and developing a method for detecting signals of bedridden patients requesting assistance using machine learning and the Internet of Things (IoT)

ออกแบบและพัฒนาวิธีการตรวจจับสัญญาณการร้องขอความช่วยเหลือผู้ป่วยติดเตียง เพื่อช่วยให้ผู้ป่วยได้รับการช่วยเหลืออย่างรวดเร็วและช่วยลดความเสี่ยงในการเกิดเหตุการณ์ร้ายแรงที่อาจส่งผลกระทบถึงชีวิต โดยการนำเทคโนโลยีอินเทอร์เน็ตของทุกสรรพสิ่ง (IoT) และการเรียนรู้ของเครื่อง (Machine Learning) มาประยุกต์ใช้ในการพยากรณ์การร้องขอความช่วยเหลือต่าง ๆ ของผู้ป่วยติดเตียง
